# Actividades evaluables de Servicios y Procesos.

Segundo año CFGS Desarrollo de Aplicaciones Multiplataforma.
Lenguaje de programación: JAVA (IDE Eclipse - Oracle).

AE1 - Actividad de introducción y repaso de conceptos y fundamentos.
AE2 - Programación multiproceso.
AE3 - Programación multihilo. Aplicación thread-safe.
AE4 - Programación de comunicaciones en red (cliente-servidor). Uso Socket - ServerSocket y clases de streams.
AE5 - Programación y generación de servicios en red (creación de un servidor HTTP para gestionar peticiones GET y POST e integración de un cliente de correo electrónico que permita el envío automático de mensajes).
